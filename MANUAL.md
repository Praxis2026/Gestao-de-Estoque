# Manual do Usuário - ClinStockPro

Bem-vindo ao **ClinStockPro**, seu sistema completo de gestão de estoque e ordens de serviço. Este manual foi desenvolvido para guiar você pelas principais funcionalidades do sistema.

---

## 1. Visão Geral

O ClinStockPro é uma plataforma centralizada para gerenciar materiais de consumo, movimentações de estoque (entradas e saídas), ordens de serviço de prótese com integração externa e relatórios detalhados para tomada de decisão.

## 2. Acesso e Login

Ao acessar o sistema, você será direcionado para a tela de login.
- **Login:** Utilize seu e-mail e senha cadastrados.
- **Segurança:** O sistema utiliza autenticação segura para proteger seus dados.

## 3. Dashboard

O Dashboard é sua tela inicial e oferece uma visão rápida do estado do seu estoque:
- **Resumo Mensal:** Entrada e saída de itens.
- **Alertas de Estoque:** Notificações de itens com estoque baixo ou crítico.
- **Atalhos Rápidos:** Acesso direto para criar novas movimentações ou ordens de serviço.

## 4. Gestão de Materiais

No menu **Materiais**, você pode:
- **Cadastrar Novos Itens:** Definir nome, categoria, unidade de medida e níveis de estoque (mínimo e crítico).
- **Consultar Estoque:** Ver a quantidade atual disponível de cada item.
- **Editar/Excluir:** Manter sua lista de materiais sempre atualizada.

## 5. Movimentações de Estoque

No menu **Movimentações**, você registra todas as alterações de estoque:
- **Entrada (+):** Registro de novas compras ou reposições.
- **Saída (-):** Registro de consumo de materiais.
- **Histórico:** Filtre e consulte movimentações passadas por data, material ou tipo.

## 6. Ordens de Serviço de Prótese (Nova OS)

Este módulo é específico para o controle de serviços laboratoriais.

### Funcionalidades:
- **Criação de OS:** Preencha os dados do paciente, curso e selecione o laboratório (fornecedor).
- **Busca de Preços Externa (Lupa):** 
  - Ao selecionar um laboratório, o sistema tenta conectar-se à tabela de preços externa do laboratório via CNPJ/CPF.
  - No campo de descrição do item, utilize a **lupa** ou comece a digitar para ver sugestões de itens e preços reais praticados pelo laboratório.
- **Entrada Manual:** Caso um item não seja encontrado na tabela externa, o sistema permite a entrada manual da descrição e preço para que você não fique travado.
- **Gestão de Status:** Acompanhe se o serviço está pendente, em execução ou finalizado.

## 7. Relatórios

O sistema oferece diversas visões analíticas:
- **Relatório de Materiais:** Visão geral simplificada.
- **Inventory Ageing:** Análise de idade do estoque.
- **Relatório de Próteses:** Resumo financeiro e operacional dos serviços laboratoriais.
- **Compatibilidade:** Análise de uso de componentes.

## 8. Configurações e Cadastros Base

Para o perfeito funcionamento do sistema, mantenha os cadastros base atualizados nos menus laterais:
- **Categorias:** Organize seus materiais (Ex: Ortodontia, Cirurgia).
- **Unidades:** Defina as medidas (Ex: Cx, Un, Ml).
- **Fornecedores:** Cadastre laboratórios e vendedores (O campo CPF/CNPJ é crucial para a integração de preços).
- **Pacientes/Cursos:** Vincule suas OS a contextos específicos.
- **Equivalências:** Configure como diferentes nomes de itens se relacionam.

## 9. Administração (Perfis e Usuários)

Se você for um administrador, terá acesso aos menus:
- **Perfis:** Defina o que cada tipo de colaborador pode ver ou editar.
- **Usuários:** Gerencie as contas de acesso (ativar, desativar ou resetar senhas).
